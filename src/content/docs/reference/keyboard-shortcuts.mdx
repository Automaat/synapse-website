## Command palette results

The palette searches three sources: **pages**, **tasks**, and **projects**. Select a result to navigate. Operational actions (start orchestrator, run triage, reload config, etc.) are not exposed through the palette today ‚Äî use the Settings page or the CLI.

## Customizing

Shortcuts aren't rebindable yet. Open a GitHub issue if a specific binding conflicts with your system.

## Platform notes

- **macOS**: `‚åò` is Command
- **Linux**: `‚åò` maps to Super (Meta). Some window managers eat `Super+<key>` ‚Äî rebind at the WM level if so.
- **Windows**: not officially supported. Wails runs, shortcuts mostly work, no guarantees.

title: CLI Reference
description: Complete reference for the sybra-cli command-line tool.
---

`sybra-cli` is a standalone binary for task and project CRUD. It reuses the same store and config as the GUI, making it safe to use alongside the running app.

## Global flags

| Flag | Description |
|------|-------------|
| `--json` | Output machine-readable JSON (for scripting) |

## Tasks

### `sybra-cli list`

```sh
sybra-cli list [--status STATUS] [--tag TAG]
```

List tasks. Filters are optional and combinable.

```sh
sybra-cli list --status todo
sybra-cli list --tag backend --json
```

### `sybra-cli get`

```sh
sybra-cli get <id>
```

Print a single task (frontmatter + body).

### `sybra-cli create`

```sh
sybra-cli create --title TITLE [options]
```

| Flag | Default | Description |
|------|---------|-------------|
| `--title` | required | Task title |
| `--body` | `""` | Markdown body |
| `--mode` | `headless` | `headless` or `interactive` |
| `--tags` | `[]` | Comma-separated tags |
| `--project` | `""` | `owner/repo` project ID |

```sh
sybra-cli create \
  --title "Add rate limiting middleware" \
  --mode headless \
  --tags backend,infra \
  --project myorg/api
```

### `sybra-cli update`

```sh
sybra-cli update <id> [options]
```

| Flag | Description |
|------|-------------|
| `--title` | New title |
| `--status` | New status |
| `--body` | New markdown body |
| `--mode` | New agent mode |
| `--tags` | New tags (replaces existing) |

```sh
sybra-cli update task-abc123 --status in-review
```

### `sybra-cli delete`

```sh
sybra-cli delete <id>
```

Permanently deletes the task file. Running agents are not stopped automatically.

## Projects

### `sybra-cli project list`

List all registered projects.

### `sybra-cli project get <id>`

Print project metadata.

### `sybra-cli project create`

```sh
sybra-cli project create --url https://github.com/owner/repo [--type TYPE]
```

Clones the repo as a bare repository and registers it.

### `sybra-cli project delete <id>`

Remove a project registration and its bare clone.

## Exit codes

| Code | Meaning |
|------|---------|
| `0` | Success |
| `1` | General error |
| `2` | Not found |

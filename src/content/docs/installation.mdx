## Requirements

- macOS 13+ (GUI mode)
- [Claude Code](https://github.com/anthropics/claude-code) installed and authenticated (`claude` on `$PATH`)
- Git 2.30+ (for worktree support)

## Download

Grab the latest release from [GitHub Releases](https://github.com/automaat/sybra/releases).

```sh
# macOS — download and move to Applications
curl -Lo Sybra.dmg https://github.com/automaat/sybra/releases/latest/download/Sybra.dmg
open Sybra.dmg
```

## Build from source

```sh
# Prerequisites: Go 1.26+, Node 24, mise
git clone https://github.com/automaat/sybra
cd sybra
mise install

# Build frontend then app
cd frontend && npm install && npm run build && cd ..
wails build
```

## CLI tool

The `sybra-cli` binary is included in every release. Install it separately for scripting:

```sh
go install github.com/automaat/sybra/cmd/sybra-cli@latest
```

## Server / Docker

```yaml
# docker-compose.yml
services:
  sybra:
    image: ghcr.io/automaat/sybra:latest
    volumes:
      - /data/sybra/home:/root/.sybra
      - /data/sybra/claude:/root/.claude
    ports:
      - "8080:8080"
    restart: unless-stopped
```

See [Server Deployment](/docs/server-deployment/) for Traefik + TLS setup.

## First run

On first launch, Sybra creates `~/.sybra/` with:

```
~/.sybra/
├── config.yaml      # main config
├── tasks/           # your task files
├── projects/        # project metadata
├── clones/          # bare git repos
└── worktrees/       # per-task checkouts
```

Edit `~/.sybra/config.yaml` to configure GitHub tokens, Todoist integration, and project types.
